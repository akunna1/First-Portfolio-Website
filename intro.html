Introductory Terminologies

What is coding? It is the process of translating human-readable instructions into a machine-executable format, typically using a programming language, to instruct computers to perform specific tasks or functions.

What is programming? It encompasses the entire process of developing software, which includes designing, planning, coding, testing, debugging, and maintaining software applications. It involves the higher-level tasks of architectural design, problem-solving, and algorithm development.

What is Web development? It is the process of creating and maintaining websites and web applications, encompassing tasks such as designing, coding, and optimizing to ensure functionality and user experience on the internet.

Key Definitions ðŸ”‘:

• WYSIWYG: A type of editing software that allows users to see and edit content in a form that appears as it would when displayed on an interface, webpage, slide presentation, or printed document. WYSIWYG is an acronym for "what you see is what you get."

• HTML: HTML (Hypertext Markup Language) is a standard markup language used to structure and present content on the World Wide Web. It uses a set of tags to define the elements of a web page, such as headings, paragraphs, links, images, and more.

• CSS:(Cascading Style Sheets) is a stylesheet language used to define the visual presentation and layout of HTML elements on a web page, enabling designers to control aspects like fonts, colors, spacing, and responsive design.

• JavaScript: A versatile programming language commonly used for adding interactivity and dynamic behavior to web pages, allowing developers to create responsive and engaging user experiences.

Introduction to Data Science

Introduction to Mobile App Development

What is Mobile App Development? It is the creation and maintenance of applications tailored for mobile devices, focusing on design, coding, and optimization to deliver a user-friendly experience on smartphones and tablets.

I specialize in the making of iOS apps. To make them, I use Swift programming language and Xcode for the IDE i.e Integrated Development Environment. Android apps, on the other hand, are typically developed using Kotlin programming language and Android Studio for the IDE. How about Flutter? Heard of it? 😁

- What was used to make iOS apps before Swift and Xcode IDE? Before Swift, Objective-C dominated iOS app development, still in use alongside Xcode IDE. Apple's 2014 Swift introduction revolutionized iOS, bringing a safer, concise, and high-performance language.

- What was used to make Android apps before Kotlin and Android Studio IDE? Android app development was primarily done using the Eclipse IDE with the Android Development Tools (ADT) plugin. This combination facilitated programming in Java for Android app development.
